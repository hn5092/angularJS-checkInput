本框架是基于angular双向绑定和指令实现的,通过angular的双向数据绑定和指令的特性,实现多个指令数据
的共享,从而达到无需写任何JS代码便可以进行表单验证的功能
V1.0:
1.实现多个标签之间的数据共享
2.通过cs属性里面设置  "限制字符长度|功能 功能"实现功能的添加,其中功能选项可以是多个,通过
空格隔开,现在默认大写W是文本 N是数字,NN是notNULL ,其中JS方法中并未实现判断方法体,默认返回TRUE,
使用时应去写自己需要的方法体来实现
3.添加判断功能,只需要在check_main中添加所需方法,然后在check函数中调用即可,如需自定义提示内容
可以在initialize中初始化提示信息里面添加即可
4.自定义选项将在不久之后进行调整,通过读取配置文件方式搭配.
5.返回值是scope.rs

新功能预告:
1.在茫茫页面中无数个表单中可以实现分组效果, 自定义所需分组,只需要一个数字即可实现. 预计一周内上线
2.提示信息准备搭配bootstrap,加入多种显示效果供用户选择
3.扩展数据共享功能实现一些更为高级的功能.(据说可以把所有表单智能化哦!~)
