本框架是基于angular双向绑定和指令实现的,通过angular的双向数据绑定和指令的特性,实现多个指令数据 的共享,从而达到无需写任何JS代码便可以进行表单验证的功能 V1.0:
1.实现多个标签之间的数据共享

2.通过cs属性里面设置 "限制字符长度|功能 功能"实现功能的添加,其中功能选项可以是多个,通过 空格隔开,现在默认大写W是文本 IN是整数,NN是notNULL ,

3.添加判断功能,只需要在check_main中添加所需方法,然后在check函数中调用即可,如需自定义提示内容 可以在initialize中初始化提示信息里面添加即可(未来加入配置文件方式,提高插件扩展性)

4.每一种 验证都自动对应一种类型的提示,若不满足某一条件都会进行提示(默认为追加span 标签 后期优化加入BOOTSTRAP的悬浮提示 并可提供多种选择)

5.自定义选项将在不久之后进行调整,通过读取配置文件方式搭配.

6.返回值是scope.rs() 默认无参数是全局变量,参数是分组信息

新功能预告:
1.在茫茫页面中无数个表单中可以实现分组效果, 自定义所需分组,只需要一个数字即可实现. 预计一周内上线(已经实现,具体看DEMO)

2.提示信息准备搭配bootstrap,加入多种显示效果供用户选择

3.准备对其他type的input加入验证入radio select的非空检测等

4.扩展数据共享功能实现一些更为高级的功能.(据说可以把所有表单智能化哦!~)
